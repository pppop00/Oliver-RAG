# 教育背景 (Education)

这个文件夹记录所有正式的教育经历和学习经历。

## 内容类型
- ✅ **学位教育**：高中、本科、硕士、博士
- ✅ **在线课程**：Coursera, Udacity 等完整的认证课程
- ✅ **训练营**：编程训练营、数据科学营等
- ✅ **专业证书**：专业认证和资格证书
- ✅ **交换学习**：海外交换或访学经历

## 文件命名规范
```
{institution-abbreviation}-{degree-type}-{major}.md
```

示例：
- `stanford-bs-computer-science.md`
- `mit-ms-data-science.md`
- `coursera-deeplearning-specialization.md`

## 使用模板
复制 `../_templates/education-template.md` 开始创建新的教育文档。

## 关键要素
每个教育文档应包含：
1. 学校、专业、学位基本信息
2. GPA、排名、荣誉等学术表现
3. 核心课程列表和学习收获
4. 毕业论文或重点项目
5. 课外活动和学生组织
6. 期间的实习和工作经历
7. 重要导师和人际网络
8. 对职业发展的影响

## 教育层次
按时间顺序或重要性排列：
- 高中教育
- 本科教育
- 研究生教育
- 继续教育/专业发展

## RAG 优化提示
- 详细列出核心课程和主要收获
- 链接到该时期完成的所有项目
- 记录掌握的技能及其应用
- 量化学术成就（GPA、排名、获奖）
- 保存课程大纲、成绩单等结构化数据
