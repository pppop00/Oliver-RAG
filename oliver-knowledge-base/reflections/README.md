# 个人反思 (Reflections)

这个文件夹存放个人思考、学习笔记、价值观探索和深度反思。

## 内容类型
- ✅ **项目反思**：项目完成后的复盘和总结
- ✅ **技能学习**：学习某个技能的心得体会
- ✅ **职业发展**：职业选择和发展的思考
- ✅ **价值观探索**：个人价值观和人生目标的反思
- ✅ **失败经验**：失败的项目或经历的教训
- ✅ **阶段性总结**：年度、季度或重要节点的总结

## 文件命名规范
```
{date}-{topic-in-kebab-case}.md
或
{topic-in-kebab-case}.md  # 对于长期更新的反思
```

示例：
- `2024-q4-career-reflection.md`
- `learning-machine-learning-journey.md`
- `first-year-as-software-engineer.md`
- `values-and-goals.md`

## 使用模板
复制 `../_templates/reflection-template.md` 开始创建新的反思文档。

## 反思类型
在 frontmatter 中标注：
- `learning`：学习相关的反思
- `career`：职业发展相关
- `personal`：个人成长和价值观
- `project`：项目复盘
- `skill`：技能掌握的思考

## 关键要素
每篇反思应包含：
1. 清晰的背景和情境
2. 客观的观察和现象描述
3. 深入的分析和洞察
4. 经验教训总结
5. 可行动的建议
6. 长期影响和价值

## 反思的价值
- 📝 记录成长轨迹
- 🧠 加深理解和内化
- 🔄 避免重复犯错
- 💡 发现模式和规律
- 🎯 明确未来方向

## RAG 优化提示
- 连接具体的项目、技能或经历
- 使用结构化的框架（如 STAR、5W1H）
- 记录情感和心理变化
- 标注关键的认知转变时刻
- 定期回顾和更新长期反思文档

## 隐私提示
⚠️ 反思可能包含敏感的个人想法。如果将知识库用于 RAG 应用，注意控制反思内容的访问权限。
