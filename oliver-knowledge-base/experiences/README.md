# 工作与经历 (Experiences)

这个文件夹存放所有职业经历和重要经验，包括：

## 内容类型
- ✅ **全职工作**：正式的全职职位
- ✅ **实习经历**：实习和暑期工作
- ✅ **兼职工作**：在校或业余兼职
- ✅ **志愿者经历**：有意义的志愿服务
- ✅ **创业经历**：创业项目和经验
- ✅ **研究经历**：实验室研究工作（非课程项目）
- ✅ **竞赛活动**：重要的竞赛参与经历

## 文件命名规范
```
{company-name}-{position-title}.md
```

示例：
- `google-software-engineer.md`
- `meta-swe-intern-2022.md`
- `stanford-research-assistant.md`

## 使用模板
复制 `../_templates/experience-template.md` 开始创建新的经历文档。

## 关键要素
每个经历文档应包含：
1. 完整的 frontmatter 元数据
2. 2-3句话的摘要
3. 职责和成果（最好量化）
4. 使用的技术和技能
5. 关联的项目链接
6. 个人成长和收获

## RAG 优化提示
- 使用具体的数字和指标（提升X%，节省X小时）
- 明确标注时间线和持续时长
- 标注所有相关技术栈和工具
- 链接到相关的项目和技能文档
